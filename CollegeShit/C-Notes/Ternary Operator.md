aka **Conditional Operator** or **Short-Hand If**

```c
"simple if-else"

int time = 20;  
if (time < 18) {  
  printf("Good day.");  
} else {  
  printf("Good evening.");  
}

OUTPUT: Good evening
```

```c
"with Ternary Operator"

int time = 20;  
(time < 18) ? printf("Good day.") : printf("Good evening.");

OUTPUT: Good evening
```
